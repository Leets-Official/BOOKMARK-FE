## ✍ 프로젝트 소개 
**나의 인사이트, 잊어버려도 쉽게 찾아낼 수 있어야 하니까**

URL만 입력하면 자동으로 썸네일과 제목을 추출해 손쉽게 북마크를 생성합니다.

카테고리와 태그를 설정해 원하는 자료를 빠르게 필터링하고 검색할 수 있습니다.

효율적인 정보 관리로 필요한 순간에 원하는 북마크를 바로 찾아드립니다.
